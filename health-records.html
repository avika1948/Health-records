⚙️ הגדרות ראשוניות נדרשות

לפני השימוש באפליקציה, יש למלא את פרטי Google Sheets שלך:

  1. עבור ל-Google Cloud Console → צור פרויקט חדש
  2. הפעל Google Sheets API + Google Drive API
  3. צור Credentials → OAuth 2.0 Client ID → Web Application → הוסף ל-Authorized origins את הכתובת שלך
  4. צור Google Sheet חדש ו-שמור את ה-ID שלו (מה-URL)
  5. בשורה 1 של ה-Sheet הוסף כותרות: תאריך | תחום | פרטים | שם התרופה | שם הרופא
